# Test Driven Hardening (TDH) - Research Framework

A methodology that brings scientific rigor and Test-Driven Development principles to security hardening through distributed LLM consensus.

## ğŸ§ª What is Test Driven Hardening?

Test Driven Hardening (TDH) is a security remediation framework where every fix must be:
1. **Preceded** by a failing test that demonstrates exploitability
2. **Followed** by the same test verifying the mitigation
3. **Validated** through distributed consensus of multiple LLMs
4. **Documented** in a self-contained, evidence-based pull request
